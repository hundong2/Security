# DH 키 교환

## 군론

### 군

- 원소의 집합
- 각 원소에 대해 정의된 특수한 이항 연산 ( + 또는 x )  
- 집합과 연산이 어떠한 속성을 만족하면 군이 된다.  
- `DH` 는 곱셈군(multiplicative group). 
- 군의 네가지 속성
  - 닫힘
  - 결합
  - 항등원
  - 역원

1) 닫힘(Closure)
집합 안에서 계산이 끝나야 합니다. (계산 결과가 집합 밖으로 튀지 않음)

의미(직관): “(G)의 원소 두 개로 연산하면 결과도 (G)의 원소다.”

2) 결합법칙(Associativity)
**괄호 위치(계산 순서)**를 바꿔도 결과가 같아야 합니다.

의미(직관): “먼저 (a*b)를 하든, 먼저 (b*c)를 하든 최종 결과가 동일.”

3) 항등원(Identity element)
아무 영향을 주지 않는 “중립 원소”가 존재해야 합니다.

의미(직관): “어떤 (e)를 붙여도 (a)가 그대로다.”
예:
덧셈에서 항등원은 (0): (a+0=a)
곱셈에서 항등원은 (1): (a\cdot1=a)

4) 역원(Inverse element)
모든 원소는 “되돌리는 원소(역원)”를 가져야 합니다.

의미(직관): “(a)를 한 뒤 (a^{-1})를 하면 항등원 (e)로 돌아간다.”
예:
덧셈에서 역원은 (-a): (a + (-a) = 0)
곱셈에서 역원은 (a^{-1}): (a \cdot a^{-1} = 1) (단, (a\neq 0) 같은 조건이 필요할 수 있음)
